## Go Comments

 A comment is a text in a code which is ignored upon the execution of that code. It is used to explain a part of a code, it can also be used to mark a part of a code that is meant to be later edited. It is also use to prevent a code from being executed.

 There are two types of comments, which are;
 - Single line comments
 - Multi-line comments

 Single-line comments: This is a type of comment used to mark a single line of text. it is usually marked with two foward slash symbols (//).
 Example:
  //This is a comment.

 Multi-line Comments: This is the opposite of a single line comment. It is used to mark multiple lines of a text. A multi-line comment starts with a /* symbol and ends with a */ symbol.
 Example:
 /*This is a multi-line
 comment*/
