## Go Output

 In Go, we have three functions to output text in a program;
 - Print()
 - Println()
 - Printf

 Print() :
 A Print() function prints its arguments with their default format.

 Println() :
 A Println() function is nothing different from the Print() funtion, only that a whitespace is added between the arguments and a new line is added at the end.

 Printf() :
 The Printf() function first formats its argument based on the given formatting verb and then prints them. There are two types of formatting verbs;
 - %v it is used to print the value of an argument
 - %t it is used to print the type of an argument
