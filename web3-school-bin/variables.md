 ## Go Variables

 A variable is a container used to store values.

 There are different types of Go variables, namely;
 - string
 - int (integers)
 - float32 (64 bases on your system's bit size)
 - bool (boolean)

 String: A string is a variable used to store text based programs. A string value is always surrounded by a double quote.

 Int (Integers) : Integers are digit based variables. Such as 123 or -123. An integer can hold both positive and negative numbers.

 float32: A float is a variable that stores numbers with decimal points. It can store both positive and negative numbers.

 Bool (Boolean): A boolean stores a value with two states; true or false.

  You can declare a variable with the var keyword e.g "var variablename type = value"
